# ♟ Chess Bot - İnsan vs Stockfish

Stockfish yapay zekası ile satranç oynayabileceğiniz Python uygulaması. Farklı ELO seviyeleri ile zorluk ayarlı oyun deneyimi sunuyor.

## 🎮 Özellikler

- **📊 400-3200 ELO Aralığı**: Sürgü ile bot seviyesi seçimi
- **💬 Bot Sohbeti**: Chess.com tarzında bot mesajları
- **🎯 4 Zorluk Seviyesi**:
  - 🟢 Başlangıç (400-1000 ELO)
  - 🟡 Amatör (1000-1600 ELO)
  - 🔴 İleri (1600-2200 ELO)
  - ⚫ Uzman (2200-3200 ELO)
- **🖥️ Tkinter GUI**: Modern ve kullanıcı dostu arayüz
- **📝 Hamle Geçmişi**: Tüm hamleleri kayıt altına al
- **↶ Geri Al**: Hamleyi geri al ve yeniden oyna
- **⏱️ Zamanı Kontrollü Hareket**: Bot'un düşünme süresi ayarlı
